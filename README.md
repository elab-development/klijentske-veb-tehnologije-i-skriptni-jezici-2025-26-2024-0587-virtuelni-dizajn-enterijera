# Interiorly

Web aplikacija za virtuelni dizajn enterijera, projekat u okviru predmeta Klijentske veb tehnologije i skriptni jezici. Omogućava uređenje prostora u 2D-u, izbor nameštaja i boja, odabir ideja.

## Pokretanje projekta
-npm install
-npm run dev

## Tehnologije

- React
- TypeScript
- Vite
- React Router
- HTML
- localStorage
- CSS

## Funkcionalnosti

- Prijava i registracija korisnika sa imenom, emailom i lozinkom
- Čuvanje korisničkih podataka u localStorage
- Prikaz korisničkog profila
- Uređivanje profila (ime i email)
- Navigacija između stranica (Početna, Ponuda, Galerija, Podaci)
- Pregled galerije ideja i čuvanje samih po želji
- Uređivanje i dizajniranje prostora

## Dizajn
Cela aplikacija je responzivna i prilagodjena mobilnim uredjajima. koriste se moderni UI elementi, hover efekti i kartice za prikaz sadržaja. Takodje korisnici mogu videti kako će im nameštaj i raspored nameštaja izgledati u sobi. Uz to mogu i menjati same dimenzije sobe i praviti ih po svojim merama.

## Čuvanje podataka
-podaci se čuvaju lokalno u browser-u pomoću localStorage.
